Установка:
==================

Необходимо установить webpack
```
npm install --global webpack
```

Установить окружение из package.json
```
npm install
```

Запуск сборки:
==================
Выполнить компанду
```
 npm start
```
И открыть в браузере http://127.0.0.1:8080
